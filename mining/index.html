Solo Mining

Step 1

Download the source code from the official Github Repository.

Step 2

Compile the source code using the instructions in the README.md file for Linux/Mac/Windows.

Step 3

Run the solominer in build/src/solominer with the following command

solominer --address <your address> --log-level 3
