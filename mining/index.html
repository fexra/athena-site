Explorer Wallet Mining Nodes
We recommend solo mining, however, shared pools are supported and may be available.

Solo Mining Instructions

Step 1 Download the source code from the official Github Repository.


Step 2 Compile the source code using the instructions in the README.md file for Linux/Mac/Windows.


Step 3 Execute ..athena/build/src/Athena with the following command:


Athena --log-level 2


Step 4 Execute ../athena/build/src/solominer with the following command:


solominer --address <your address> --log-level 3


Solo Pool Mining Instructions

Step 1 Download the athena-pool source code from the official Github Repository.


Step 2 Follow the instructions in the README.md file.


Step 3 Once everything is working, configure your pool fees to 100%.


Step 4 Point your miner to your solo pool like you would do with any other pool.
